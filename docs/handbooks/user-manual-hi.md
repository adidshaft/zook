## Zook पर कौन क्या करता है

Zook वह software है जिस पर पूरा gym चलता है — front desk से लेकर आपके फोन पर दिखने वाले workout plan तक। जो भी इसे use करता है, उसका एक role होता है — कुल छह roles हैं:

| Role | एक लाइन में | कहां काम करते हैं |
| --- | --- | --- |
| **Member** | Gym join करता है, check in करता है, plan follow करता है, shopping करता है। | Zook mobile app |
| **Owner** | Gym को business की तरह चलाता है — pricing, staff, पैसा, growth। | Web dashboard + mobile owner view |
| **Admin** | Owner की daily operations में मदद करता है, कुछ money/billing permissions कम होती हैं। | Owner जैसा ही |
| **Receptionist** | Front desk संभालता है — check-in, payment, pickup। | Desk web app या mobile |
| **Trainer** | Members को coach करता है — plans, notes, progress। | Mobile trainer view या web coach view |
| **Platform operator** | Zook का employee जो पूरे system को healthy रखता है। | Internal platform console |

अगर पक्का न हो कि आप कौन से role में हैं, तो app खोलकर देखें: scan button और shop tab दिखे तो आप **Member** हैं; revenue chart दिखे तो आप **Owner** या **Admin** हैं; check-in queue दिखे तो आप **Receptionist** हैं; client list दिखे तो आप **Trainer** हैं।

## Sign In कैसे करें

Role चाहे कोई भी हो, sign in करने का तरीका सबका एक जैसा है।

1. Zook app खोलें (या वह web page जो sign in मांगे)।
2. अपना phone number या email address डालें।
3. Zook आपको SMS या email से एक one-time code (**OTP**) भेजता है। उसे टाइप करें।
4. iPhone या Android पर, अगर option दिखे तो आप **Google** या **Apple** से भी sign in कर सकते हैं — इससे OTP step पूरी तरह skip हो जाता है।

बस इतना ही — password की जरूरत नहीं। अगर एक मिनट में code न आए, तो **Resend** दबाएं। फिर भी न आए, तो check करें कि phone/email सही टाइप हुआ है, और [Help कैसे लें](#help-कैसे-लें) section देखें।

**Password क्यों नहीं?** भूलने के लिए कम चीज़ें, और आपके फोन के बिना किसी और के लिए account में घुसना मुश्किल होता है।

## Member के लिए Guide

अगर आप gym जाते हैं और Zook app से check in करते हैं, workout plan follow करते हैं, या gym की shop से कुछ खरीदते हैं — यह section आपके लिए है।

### शुरुआत के 5 steps

1. **Sign in करें** (ऊपर देखें)।
2. **अपना gym ढूंढें** — Home tab पर **Find gyms** दबाएं, या नाम/city से search करें।
3. **Gym का page खोलें** — यहां आपको उसकी photos, plans, trainers और location दिखेंगे।
4. **एक membership plan चुनें** और pay करें (या अगर gym को पहले approval चाहिए, तो join request भेजें और approve होने का wait करें)।
5. अगली बार gym जाएं तो **Scan** tab से **check in करें**।

### App के नीचे मौजूद पांच tabs

| Tab | यहां क्या मिलेगा |
| --- | --- |
| **Home** | आपकी membership status, कितने दिन बचे हैं, और आगे क्या करना है (renew, आज का workout complete करना, वगैरह) |
| **Scan** | QR code scan करके gym में check in करें, या camera काम न करे तो code type करें |
| **Plans** | आपका workout, diet, या habit plan, अगर gym या trainer ने assign किया हो |
| **Shop** | Gym जो products बेचता है — supplements, gear, merchandise — front desk पर pickup के साथ |
| **Profile** | आपकी account details, privacy settings, language, और support |

### Gym join करना और pay करना

1. Gym का page खोलें और जो plan चाहिए उसे tap करें।
2. अगर आपके पास **referral code** या **coupon** है, तो अभी डालें — discount pay करने से पहले apply होता है।
3. Final amount ध्यान से check करें। इसमें plan price, कोई discount, और total साफ-साफ दिखना चाहिए।
4. Gym जो भी payment method offer करे उससे pay करें (card, UPI, netbanking, या कुछ मामलों में desk पर cash)।
5. अगर plan automatically renew होता है, तो आपको **autopay** का option मिलेगा — अगर हर महीने manually renew करने की चिंता नहीं चाहिए तो इसे on कर दें।
6. Payment successful होते ही आपकी membership तुरंत active हो जाती है। यह आपको Home tab पर दिख जाएगा।

**कुछ खास cases:**
- **Gym को approval चाहिए:** payment सीधे करने के बजाय, आप join request भेजते हैं। Gym का owner या receptionist उसे review करता है, approve होने पर आपको notify किया जाएगा।
- **Invite-only gym:** plans देखने के लिए भी पहले invite या referral code चाहिए होगा।
- **आपकी उम्र 18 साल से कम है:** membership शुरू होने से पहले parent या guardian की consent चाहिए होगी। App आपको इसमें guide करेगा।

**अगर payment fail हो जाए:** चिंता न करें, जो काम नहीं हुआ उसके लिए पैसे नहीं कटते। फिर से try करें, या कोई और payment method use करें। अगर पैसे कट गए लेकिन membership active नहीं हुई, तो gym से contact करें — देखें [Help कैसे लें](#help-कैसे-लें)।

### Gym में check in करना

यह वह काम है जो आप सबसे ज्यादा करेंगे, इसलिए इसे fast बनाया गया है।

1. **Scan** tab खोलें।
2. Gym के entrance पर लगे QR code पर camera point करें।
3. तुरंत इनमें से एक result दिखेगा:
   - ✅ **Approved** — check in हो गया, अंदर जाइए।
   - ⏳ **Pending** — desk को पहले कुछ double-check करना है (यह rare होता है)। Receptionist को अपना phone दिखाएं।
   - ❌ **Rejected** — कुछ गलत है (membership expire हो गई, गलत gym, वगैरह)। App बताएगा क्यों, और क्या करना है।

**अगर camera scan न करे** (कम light, camera permission न दी हो, या connection slow हो), तो **Enter code manually** दबाएं और entrance पर दिखाया गया code type करें। यह बिल्कुल वैसे ही काम करता है।

### अपना plan follow करना

अगर gym या trainer ने आपके लिए workout, diet, या habit plan बनाया है:

1. **Plans** tab खोलें।
2. आपको आज के exercises, habits, या meals एक-एक करके दिखेंगे।
3. जैसे-जैसे आप कुछ complete करें, उसे done mark करने के लिए tap करें — progress automatically save होता है।
4. Workout के दौरान, अगर किसी exercise में rest period है, तो app sets के बीच एक countdown timer दिखाता है ताकि आपको पता रहे कि अगला set कब शुरू करना है।
5. आप समय के साथ अपनी progress भी देख सकते हैं — आप कितने consistent रहे हैं, और (अगर weight जैसी चीज़ें track कर रहे हों) trend दिखाने वाला एक simple chart।

**अपनी तरफ से exercise add करना:** अगर आपने कुछ extra किया जो plan में नहीं था, तो उस दिन के लिए एक custom exercise add कर सकते हैं। यह सिर्फ उस दिन के लिए save होता है और अगली बार history check करने पर दिखता है।

### Shop का इस्तेमाल

Gym app से सीधे supplements, gear, या merchandise बेच सकता है।

1. **Shop** tab खोलें और जो चाहिए उसे browse या search करें।
2. Items को cart में add करें।
3. Checkout करें और pay करें।
4. आपको एक **pickup code** मिलेगा — order collect करने जाते समय यह front desk को दिखाएं। वे इसे scan या type करके handover confirm करेंगे।

जब तक desk आपको आपका item असल में hand over नहीं करता, तब तक कुछ भी "picked up" mark नहीं होता, इसलिए order मिले बिना charge होने का कोई risk नहीं है।

### Notifications और आपकी Profile

- **Notifications** आपको ऐसी चीज़ों के बारे में बताती हैं जैसे आने वाली renewal, trainer का नया message, या shop order जो pickup के लिए ready है।
- **Profile** में आप अपना account manage करते हैं: phone/email update करना, app की language बदलना, light और dark appearance के बीच switch करना, और जरूरत पड़ने पर support तक पहुंचना।
- **आपका data आपका है।** Profile से आप कभी भी अपने data का export मांग सकते हैं, या अपना account delete करने का request कर सकते हैं।

### अक्सर पूछे जाने वाले सवाल (Members)

**सवाल: मैंने phone बदल लिया। अपने account में वापस कैसे जाऊं?**
जवाब: बस उसी phone number या email से फिर से sign in करें — आपकी membership और history आपके account से जुड़ी है, device से नहीं।

**सवाल: क्या मैं एक से ज्यादा gym का member बन सकता हूं?**
जवाब: हां। कई gyms join करने के बाद, आप अपनी Profile से उनके बीच switch कर सकते हैं।

**सवाल: अगर workout के बीच में मेरी membership expire हो जाए तो क्या होगा?**
जवाब: session के बीच में कुछ नहीं बदलता — लेकिन renew किए बिना आप फिर से check in नहीं कर पाएंगे।

**सवाल: मैं अपना phone number gym के साथ share नहीं करना चाहता।**
जवाब: Gym सिर्फ उतना ही देखता है जितना आपकी membership चलाने के लिए जरूरी है (नाम, renewal reminders के लिए contact info, attendance)। बाकी सब आप अपनी privacy settings से control करते हैं।

**सवाल: मेरा referral code काम नहीं कर रहा।**
जवाब: Referral codes आमतौर पर pay करने *से पहले* डालने होते हैं, बाद में नहीं। spelling दोबारा check करें, और देखें कि code expire तो नहीं हुआ।

## Gym Owner के लिए Guide

अगर आप Zook पर अपना gym चलाते हैं, तो यह आपकी day-to-day operating guide है।

### आप किन चीज़ों के incharge हैं

Owner होने के नाते, Zook पर आपका gym कैसे चलता है इसका पूरा control आपके पास है: आपके gym का public profile, membership plans और pricing, staff और उनकी permissions, payments और refunds, shop, reports, और Zook के साथ आपकी अपनी billing।

### पहली बार gym set up करना

Members के लिए दरवाजे खोलने से पहले, ये चीज़ें एक बार, इसी क्रम में करें:

1. **Sign in करें** और अपना dashboard खोलें।
2. **Gym profile भरें** — नाम, logo, cover photo, address, और छोटा description। यह पहली चीज़ है जो होने वाले members देखते हैं।
3. **Branches add करें** — अगर एक से ज्यादा location है, तो हर एक को उसके अपने address और city के साथ set up करें।
4. **Membership plans बनाएं** — हर एक को साफ नाम, price, और duration दें (जैसे "Monthly", "₹1,500", "1 month")। Members यहां जो भी set up किया है उसी में से चुनते हैं।
5. **अपना staff add करें** — receptionists, trainers, और admins को invite करें, और हर किसी को सही role दें ताकि उन्हें सिर्फ जरूरी चीज़ें ही दिखें।
6. **Payment setup दोबारा check करें** — असली members से पहले renewal flow को end-to-end test कर लें।
7. **अपना public gym page preview करें** — इसे phone और desktop दोनों पर खोलकर देखें कि वह वैसा ही दिखे जैसा आप चाहते हैं।

### आपकी daily routine

ज्यादातर owners दिन में एक-दो बार Zook check करते हैं। यहां देखने लायक चीज़ें हैं:

1. **Today** view खोलें — यह उन सब चीज़ों का summary है जिन पर अभी आपका ध्यान चाहिए।
2. **Pending approvals review करें** — नए members जो join होने का wait कर रहे हैं, अगर आपके gym को approval चाहिए।
3. **Revenue और payments check करें** — देखें क्या आया, और कोई payment fail तो नहीं हुई।
4. **आने वाली renewals देखें** — ताकि members के lapse होने से पहले उन्हें nudge कर सकें।
5. **Attendance check करें** — कुछ भी unusual, जैसे कोई member जो लंबे समय से नहीं आया।
6. **अपनी shop check करें** — stock levels, और pickup के लिए ready कोई orders।
7. Members को notification तभी भेजें जब सच में उपयोगी हो — reminder, announcement, offer। किसी को शोर पसंद नहीं।

### पैसा और memberships — कुछ जरूरी habits

- **Pay करने का रास्ता छोटा रखें।** Members को एक plan, उसकी price, और एक "Pay" button दिखना चाहिए — screens की भूल-भुलैया नहीं।
- **Plans के नाम साफ रखें।** "Plan A", "Plan B" से "Monthly", "Quarterly", "Annual — 2 महीने free" ज्यादा अच्छा पढ़ने में लगता है।
- **Autopay offer करें।** इससे उन members की संख्या कम होती है जो accidentally अपनी membership lapse कर देते हैं — उनके लिए भी अच्छा, आपके revenue के लिए भी अच्छा।
- **Refunds और cancellations एक ऐसी policy follow करें जो आपने पहले से decide कर रखी हो,** ताकि हर staff member उन्हें एक जैसे तरीके से handle करे।

### आपकी team

Dashboard के Staff section से आप receptionists, trainers, और admins को invite कर सकते हैं। हर role को अलग tools का set दिखता है:

- **Admins** लगभग वह सब कर सकते हैं जो आप कर सकते हैं, कुछ sensitive money और billing actions को छोड़कर।
- **Receptionists** front desk पर check-ins, payments, और shop pickups संभालते हैं।
- **Trainers** अपने assigned members के plans और progress manage करते हैं।

कोई भी sensitive action जो कोई करता है — refund, plan change, staff permission change — आपके **audit log** में record होता है, तो आपके पास हमेशा एक record रहता है कि किसने क्या किया।

### Reports और growth

आपके dashboard का Reports section समय के साथ revenue, attendance, और member growth दिखाता है, ताकि आप देख सकें कि आपका gym सही दिशा में जा रहा है या नहीं। नए members लाने के लिए आप **coupons**, **offers**, और **referral rewards** भी manage कर सकते हैं।

### अक्सर पूछे जाने वाले सवाल (Owners)

**सवाल: एक member कहता है उसने pay कर दिया लेकिन उसकी membership active नहीं है।**
जवाब: अपने Payments section में payment की status check करें। अगर आपकी तरफ से successful हुआ लेकिन app में नहीं दिखा, तो इसे escalate करना बेहतर है — देखें [Help कैसे लें](#help-कैसे-लें)।

**सवाल: क्या members join कर चुकने के बाद मैं plan की price बदल सकता हूं?**
जवाब: हां, लेकिन आमतौर पर यह सिर्फ आगे के नए sign-ups और renewals को affect करता है, पुरानी price पर mid-cycle members को नहीं।

**सवाल: staff member की access कैसे हटाऊं?**
जवाब: Dashboard के Staff section से, आप किसी की भी access तुरंत deactivate कर सकते हैं।

**सवाल: Owner और Admin में क्या फर्क है?**
जवाब: आपके (Owner के) पास पूरा control है, billing और business-level settings समेत। Admins daily operations चला सकते हैं लेकिन कुछ sensitive money और account-level actions तक उनकी access नहीं होती।

## Admin के लिए Guide

Admins gym को day to day चलाने में owners की मदद करते हैं। आप एक powerful role हैं — लेकिन owner नहीं, इसलिए कुछ billing और business-control actions आपके लिए off-limits हो सकते हैं।

### आपकी daily routine

1. Sign in करें और confirm करें कि किस gym/branch पर काम कर रहे हैं।
2. Quick summary के लिए **Today** board खोलें।
3. जो भी pending member approvals हैं उन्हें clear करें।
4. Attendance में कुछ unusual तो नहीं यह check करें।
5. Payments, shop orders, और stock levels review करें।
6. जो भी staff किसी issue में फंसा हो उसकी मदद करें।
7. अगर कुछ sensitive हुआ हो, तो audit history check करके समझें क्या बदला।

### आप क्या कर सकते हैं

| Area | आप क्या कर सकते हैं |
| --- | --- |
| Members | Search, details देखना, join requests approve/reject करना |
| Plans | Owner ने permission दी हो तो plans create/update करना |
| Promotions | Coupons, offers, और referral rules manage करना |
| Attendance | कुछ भी unusual review करना, QR check-in manage करना |
| Payments | Offline (cash/manual) payment record करना, payment history देखना |
| Shop | Products, stock levels, और orders manage करना |
| Staff | Permission हो तो अन्य staff को invite/manage करना |
| Reports | Owner के review के लिए reports download करना |

### कुछ जरूरी बातें

- असली prices तब तक न बदलें जब तक owner ने specifically न कहा हो।
- Owner की approval के बिना refund न दें, जब तक gym की policy इसकी साफ इजाजत न दे।
- अगर "permission denied" message दिखे, तो कुछ sensitive actions के लिए यह normal है — owner से पूछें।
- जो भी sensitive काम आप करते हैं वह audit history में दिखता है, और यह अच्छी बात है — यह आपकी भी सुरक्षा करता है।

## Receptionist के लिए Guide

आप front desk संभालते हैं। आपका काम है fast और accurate होना: members को entry देना, उन्हें ढूंढना, payments लेना, और shop orders hand over करना।

### आपकी daily routine

1. अपनी desk screen खोलें (web पर `/desk`, या mobile reception view)।
2. Confirm करें कि किस branch पर काम कर रहे हैं।
3. Member search और check-in screens हमेशा ready रखें — इनका इस्तेमाल लगातार होगा।
4. जैसे ही members आएं, उनका check-in verify करें।
5. Amount और member दोनों double-check करने के बाद ही payment record करें।
6. Item असल में hand over करने के बाद ही shop pickup को fulfilled mark करें।
7. अगर कुछ भी गलत या unclear लगे, तो force न करें — owner या admin से पूछें।

### आपके मुख्य काम, step by step

**Member को check in करना:**
1. उनका attendance code scan करें या type करें।
2. Result देखें: approved, pending, या rejected।
3. सिर्फ तभी अंदर जाने दें जब approved दिखे। Pending या rejected हो तो gym की policy follow करें (आमतौर पर: owner/admin से पूछें, या rejection reason देखें)।

**Member ढूंढना:**
1. नाम, phone number, या code से search करें।
2. कुछ भी करने से पहले (payment record करना, सवाल का जवाब देना, वगैरह) उनकी details खोलकर membership status check करें।

**Payment record करना:**
1. Member select करें।
2. सही amount डालें।
3. Payment method चुनें (cash, card, वगैरह)।
4. Confirm करें — और confirm करने से पहले amount और member का नाम दोबारा check करें, क्योंकि इसे बाद में undo करना आसान नहीं होता।

**Shop pickup hand over करना:**
1. Ready order खोलें।
2. Member से pickup code मांगें।
3. Verify करें कि वह match करता है।
4. तभी उसे fulfilled mark करें।

### कुछ जरूरी rules

- अगर app कहे कि member की entry rejected या expired है, तो उन्हें अंदर न जाने दें — भले ही वे कहें कि यह गलती है। इसे escalate करें।
- Item member के हाथ में असल में जाने से पहले pickup को fulfilled mark न करें।
- Confirm करने से पहले हमेशा payment amount और member का नाम दोबारा check करें — गलती Confirm दबाने से पहले पकड़ना बाद में पकड़ने से बहुत आसान है।
- Confirm या Fulfil को एक से ज्यादा बार tap करने से बचें — अगर app slow लगे, तो दोबारा tap करने के बजाय wait करें।

## Trainer के लिए Guide

आप members को coach करते हैं — plans assign करना, उनकी progress track करना, और contact में रहना।

### आपकी daily routine

1. Mobile पर अपना trainer view खोलें।
2. देखें आज किन clients को आपके ध्यान की जरूरत है।
3. Session से पहले, उस client की details खोलें और उनका current plan और recent progress review करें।
4. Session के बाद, notes add करें, या session कैसा गया उसके हिसाब से plan update करें।
5. अगर AI-drafted plans use कर रहे हैं, तो publish करने से पहले हमेशा carefully review करें — इन्हें first draft समझें, final answer नहीं।
6. Confirm करें कि member अभी publish किया गया plan असल में देख पा रहा है।

### App में आपको क्या मिलेगा

| Area | किस लिए |
| --- | --- |
| Home | आज का summary और कोई clients जिन्हें ध्यान चाहिए |
| Clients | आपको assign किए गए clients search और open करना |
| Plans | Workout/diet/habit plans create, edit, assign, और publish करना |
| Notes | Notes save करना — या तो private (सिर्फ आपके लिए) या member को दिखने वाले |
| Progress | Attendance देखना, वे plan कितनी consistency से follow कर रहे हैं, और उनकी logged progress entries |
| Inbox | आपके members की तरफ से messages या check-ins |

### कुछ जरूरी rules

- आपको हमेशा सिर्फ वे clients दिखेंगे जो आपको assign हैं — यह जानबूझकर है, सबकी privacy के लिए।
- अगर plan draft करने में AI की मदद ले रहे हैं, याद रखें यह एक suggestion है, final instruction नहीं — publish करने से पहले पूरा पढ़ें।
- Medical conditions, injuries, diet, या supplements से जुड़ी किसी भी चीज़ में सावधान रहें — सिर्फ वही advice publish करें जो appropriate हो और आपके gym की policy से match करे।
- आपके notes किसी member की gym history को shape कर सकते हैं, इसलिए इन्हें clear, factual, और respectful रखें।

## Platform Operator के लिए Guide

यह section Zook की अपनी internal team के लिए है, जो platform के हर gym में पूरे system को healthy रखती है।

### आप किसके responsible हैं

आप system health, provider status (payments, SMS, email), customer escalations, और platform पर कहीं भी हो रहे किसी भी unusual चीज़ को monitor करते हैं — किसी एक gym को नहीं।

### आपके मुख्य tools

| Area | किस लिए |
| --- | --- |
| Platform console | पूरे system का overview और operational controls |
| Status/readiness | Check करना कि API, database, और third-party providers healthy हैं |
| Webhooks | Payment/provider events inspect करना, या जब safe हो तो replay करना |
| Notifications | Broadcast messages और delivery issues review करना |
| Audit | Platform पर कहीं भी हुई किसी sensitive action को investigate करना |
| Support escalation | Policy bypass किए बिना किसी खास gym की मदद करना |

### कुछ जरूरी rules

- Documented support reason के बिना कभी किसी customer की तरह impersonate न करें या उनका data modify न करें।
- Secrets — API keys, tokens, credentials — कभी भी screenshots, tickets, docs, या chat में share न करें।
- Refunds, webhook replays, production scripts चलाना, और mass notifications — इन सबके लिए action लेने से पहले साफ approval चाहिए।
- क्या किया इसका record रखें: date, environment, action, और result।
- अगर कोई provider (जैसे payment gateway या SMS service) down हो जाए, तो कोई भी setting बदलने से पहले impact और workaround document करें।

## Help कैसे लें

अगर कुछ वैसे काम नहीं कर रहा जैसा इस guide में बताया गया है:

1. **Members और trainers:** app में **Profile → Support** खोलें।
2. **Owners, admins, और receptionists:** अपने dashboard के support option का इस्तेमाल करें, या अपने Zook account contact से संपर्क करें।
3. **सबके लिए:** अगर आप साफ-साफ बता सकें — कौन सा gym और branch, आप किस role से sign in हैं, और जब चीज़ खराब हुई तब आप exactly क्या करने की कोशिश कर रहे थे — तो इससे बहुत मदद मिलती है।

अगर आप एक staff member हैं और किसी और की मदद कर रहे हैं, तो इस क्रम में सवाल पूछना अच्छा रहता है:
1. वे कौन सा gym और branch use कर रहे हैं?
2. वे किस role से sign in हैं?
3. वे क्या करने की कोशिश कर रहे थे?
4. क्या यह payment, attendance, membership, shop order, या plan से जुड़ा है?
5. क्या यह असली app में हो रहा है, या test/staging environment में?
6. अगर fix में पैसा, membership status, access, या customer data शामिल है — action लेने से पहले approval लें।
7. Resolve होने के बाद, final outcome क्या रहा यह note कर लें।

## Glossary (शब्दावली)

| Term | Meaning |
| --- | --- |
| **OTP** | One-Time Password — password की जगह sign in करने के लिए आपके phone या email पर भेजा गया एक छोटा code |
| **Check-in** | Gym पर अपने पहुंचने का record करने के लिए scan करना (या code enter करना) |
| **Autopay** | आपकी membership हर cycle में automatically renew और charge होती है, तो आपको हर बार manually pay नहीं करना पड़ता |
| **Pickup code** | पहले से pay किया हुआ shop order collect करने के लिए front desk पर दिखाया जाने वाला code |
| **Referral code** | एक code जो आप share करते हैं (या पाते हैं), जो किसी दोस्त के join करने पर discount या reward देता है |
| **Branch** | Gym की एक physical location, अगर gym की एक से ज्यादा locations हैं |
| **Audit log** | System में हुई हर sensitive action का, और किसने की, इसका record |
| **Plan** (membership) | Member द्वारा खरीदा गया एक pricing package — जैसे "Monthly", "Annual" |
| **Plan** (workout) | Trainer या gym द्वारा किसी member को assign किए गए exercises, diet guidance, या habits का सेट |
