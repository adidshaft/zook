# Zook copy style guide

Use sentence case everywhere: headings, buttons, tabs, labels, and chips.

Use plain language. Avoid abbreviations, internal operations jargon, and clever labels when a small-gym owner in India would expect a direct word.

Use one name per concept. Do not switch between similar labels for the same surface or role.

Format money with the existing `formatInr` helper. Format dates with `formatDateTime` or the app's existing short date helpers.

Rename map:

| Avoid | Use |
| --- | --- |
| Cmd | Today |
| Command view / command copy | Today / plain descriptions |
| Plans in motion | Active plan work |
| Today coaching actions | Today |
| Feature Locked | AI drafting is off |
| Trainer mode | Trainer |
| Review for owner approvals | Approvals |
| Member Profile tab for the You screen | You |
| Desk as a screen title | Front desk |
| Reception as a role name | Reception |
